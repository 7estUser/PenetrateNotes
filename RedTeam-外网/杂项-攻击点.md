# 内容编码降级攻击
`Accept-Encoding: gzip;q=0, deflate;q=0`  
原理：强制服务端返回未压缩明文  
防御：禁用不安全的编码方式

# 协议走私攻击向量
```http
GET / HTTP/1.1
Transfer-Encoding: chunked

0

GET /admin HTTP/1.1
```

防御：配置中间件拒绝歧义请求

# Null字节注入
在尝试使用特殊字符注入的时候，可以尝试Null字节。例如URL编码`%00`，转义字符`\0`  
在JSON请求中，不能直接使用常规的URL编码等，可以用\u序列插入Uonicode字符，即`\u0000`，表示Null字节字符。
